# Icon PWA & favicon

Aplikasi memakai path **`/gambar/icon/`** (bukan `/icon/`).

Pastikan file berikut bisa diakses di URL tersebut (mis. dari backend atau `public/gambar/icon/`):

- **mybeddian.png** – favicon, apple-touch-icon, prompt install PWA
- **mybeddian128.png** – 128×128 (manifest)
- **mybeddian192.png** – 192×192 (manifest)
- **mybeddian512.png** – 512×512 (manifest)
