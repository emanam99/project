# Ikon PWA (legacy)

Aset resmi aplikasi memakai seri **mybeddien** di folder `public/gambar/icon/` (disalin dari `htdocs/gambar/icon/`).

- **mybeddien.png** – favicon, apple-touch-icon, prompt install PWA
- **mybeddien128.png** – 128×128 (manifest)
- **mybeddien192.png** – 192×192 (manifest)
- **mybeddien512.png** – 512×512 (manifest)

Manifest Vite memakai path `${VITE_GAMBAR_BASE}/icon/mybeddien*.png` (default `/gambar`).
